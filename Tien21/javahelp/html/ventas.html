VENTAS

En el apartado de ventas podremos realizar las ventas.

Esta ventana consta de de una tabla a la que se le añaden los productos de la venta. Dos botones uno para añadir productos y otro para borrar los productos. Un Label que muestra el precio total de la venta y otro que muestra la devolucion. Un texto field para introducir la cantidad con la que el cliente paga y un boton para realizar la compra.

Para realizar una venta, vamos añadiendo productos a la tabla. Pulsamos en el boton añadir y buscamos el producto por nombre o codigo, cuando pulsemos OK lo añadira a la tabla de productos y modificara el precio total. Este proceso se repite tantas veces como productos tengamos

Cuando todos los productos estan añadidos en la tabla introducimos la cantidad que el cliente entrega y pulsamos el boton de realizar venta. La venta se añadira a la base de datos y se mostrara el cambio que de debe entregar al cliente el el label correspondiente.
